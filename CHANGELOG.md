# Changelog

All notable changes are documented here.

## 1.0.0 - 2026-07-18

### Added

- Native WSL bootstrap, diagnostics, and operator CLI.
- Deterministic PostgreSQL source and isolated recovery lab.
- Ansible backup, disaster, restore, and verification workflows.
- Encrypted restic snapshots with full repository integrity checks.
- Explicit destructive approval and Docker Compose scope validation.
- JSON and Markdown recovery evidence with RTO and restore metrics.
- Unit, contract, lint, and live-rehearsal GitHub Actions workflows.

### Fixed

- Bootstrap now rebuilds a stale virtual environment when the selected Python
  interpreter changes.
